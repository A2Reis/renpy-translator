# Kitsune — Ren'Py Game Translator

Translate Ren'Py visual novel games into any language before playing. Free, no API key required.

## Features

- **One-click translation** — Select game folder, choose languages, click Start
- **133 languages supported** — Any language Google Translate supports
- **Free forever** — Uses Google Translate, no API key needed
- **Smart translation** — Instant dictionary for UI terms, proper name preservation, gender-aware
- **Translation validator** — Scans for errors, fixes them automatically
- **Deep check** — Back-translation verification catches idiom errors
- **Multi-language UI** — App interface translates to your language automatically
- **Non-destructive** — Original game files are never modified (uses backup system)

## Download

Go to [Releases](https://github.com/A2Reis/renpy-translator/releases) and download the latest `Kitsune.exe`.

No Python installation required — just download and run.

## How to Use

1. Open **Kitsune.exe**
2. Click **Translate** in the sidebar
3. Select your game folder (the one with the `game/` subfolder)
4. Choose source language (usually English)
5. Choose target language (your language)
6. Choose which game language to replace
7. Click **Start Pre-scan**
8. Open the game and play in your language!
